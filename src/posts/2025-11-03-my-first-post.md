欢迎来到我的学习日志！这是我搭建个人网站后的第一篇文章。

## 今天学到了什么

通过这次搭建，我学到了：

- 学习了如何搭建个人网站
- 使用了 Eleventy 静态网站生成器
- 部署到了 Vercel
- 配置了 Netlify CMS（虽然遇到了一些波折）

## 遇到的挑战

搭建过程中遇到了不少问题：
- Netlify 超出使用限制
- CMS 认证配置复杂
- 最终选择了 Vercel + GitHub 的方案

## 收获

虽然过程曲折，但最终成功了！这让我对 Web 开发有了更深的理解。

**很有成就感！** 🎉
